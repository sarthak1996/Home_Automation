This folder contains Raspberry Pi Website Code
Use xampp to create a apache server
Paste all the files in C:.../xampp/htdocs/Folder (htdocs is located in xampp installation directory)
Start apache server and sql
Open phpmyadmin and make apt tables as shown in screenshot
Read connect.php and set password accordingly
After all is done open localhost/Folder/        (Folder is the folder where you paste all the php files)
Open login.php
